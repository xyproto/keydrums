# keydrums

Play drums with the `qweasd` keys, randomze the kit with `f` and stop the sound with `space`.

## Screenshot

![keybindings](img/keybindings.png)

## General info

* Uses SDL2.
* Tested on Arch Linux.
* Build with `make` or with [`cxx`](https://github.com/xyproto/cxx).
